# AutiTech 🚀

AutiTech is an AI-powered assistive application designed to support children with autism by improving communication through multimodal interaction.

## Features
- 🖼️ Text to Image (CLIP-based matching)
- 👦 Image to Speech (gTTS)
- 🎤 Speech to Text (Whisper)

## Tech Stack
- Streamlit
- Python
- OpenAI CLIP
- Faster-Whisper

## Live Demo
https://autitech-27.streamlit.app/#auti-tech

## How to Run
pip install -r requirements.txt
streamlit run app.py
